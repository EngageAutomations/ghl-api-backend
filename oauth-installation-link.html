Click the button below to install the OAuth app and restore product creation capabilities:

Install OAuth App

Installation Steps:

  1. Click the "Install OAuth App" button above
  2. Choose your GoHighLevel location (MakerExpress 3D recommended)
  3. Approve the permissions for product and media management
  4. You'll be redirected to your marketplace with a new installation ID
  5. The new installation will have full product creation capabilities

Included Scopes:

  • products.write - Create and modify products
  • products.readonly - Read product information
  • medias.write - Upload images to media library
  • medias.readonly - Access media files
  • locations.readonly - Access location data
  • contacts.readonly/write - Contact management
  • users.readonly - User information access

After Installation:

Your form will automatically create real products in GoHighLevel using the data you enter. The system will capture:

  • Product name and description
  • Pricing information
  • Image uploads to media library
  • Location association
Installation will be saved as: install_[timestamp]
