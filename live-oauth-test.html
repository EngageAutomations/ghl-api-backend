OAuth Flow Process

Step 1: Authorization Request

Click "Start Live OAuth Flow" to redirect to GoHighLevel's authorization server. You'll be prompted to:

  • Select a GoHighLevel location/account
  • Review the requested permissions
  • Grant authorization to the application

Step 2: Authorization Grant

GoHighLevel will redirect back to our callback endpoint with:

  • Authorization Code - Temporary code for token exchange
  • State Parameter - Security validation
  • Location Data - Selected GoHighLevel account info

Step 3: Token Exchange

Our server automatically exchanges the authorization code for access tokens:

  • Access Token - For API requests
  • Refresh Token - For token renewal
  • Token Expiry - Security management
